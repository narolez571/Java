This package is a This package contains the following:
  • helloworld - this package contains a really simple MBean interface to show how easily an MBean can be deployed to an MBean server. It uses the HttpAdapter to gain access to the Mbean server.
  • jmxbook - this package breaks down the JMX agent to have two access points: the HttpAdapter as used before; and the RMI Connector server. This is then access via an MBean server connection to manage the deployed MBeans. This implementation also deploys an MBean to the servr through the RMI Connector. This package also contains a collection of MBean components that can be deployed to the MBean server with the client:
    • propertymanager - Standard MBean that allows access to a properties file
    • dbmanager - Stanadrd MBean that allows access to a datasource through JNDI
    • logmanager - Standard MBean that allows access to a log manager class (this utilises an MBen to Mbean lookup)
    • polling - Standard MBean that uses notifications. This is a simple start/stop thread application
