Pete's Web App Sandbox from the JSP Book

In here we have a collection of dynamic content as per the JSP Book "Sevlets and Java Server pages (Wesley)"

Site Logger Configuration

Database Access

  • Create Database Schema - Creates a database schema for the local db
  • Arbitrary SQL - executes some arbitrary SQL
  • Show database tables - shows the table metadata for the tables in the database

Design patterns

  • Model 1 design pattern - an example of a small web app put together using a Model 1 design pattern
  • Model 2 design pattern - an example of a small web app put together using a model 2 design pattern

Security

  • Secured Page - this is a secured page ih the web app

Session State

  • Show Cookies - shows the current session cookies and their data
  • Edit Cookies - servlet to allow you to edit the current session cookies (create/update/delete)
  • Cookie Tracker - an example way to show you how to track user access through cookies
  • Encode URLs - shows how the session ID can be encoded into the URL

Filters

  • Link Tracker - this is an interface to the Link Tracker filter
  • Test Audit Filter - this page should test the auditing filter
  • Compression Test - this test will compare the compression filter results within the webapp, this we can see the net gain from using a compression filter.
  • Cache Filter Test - BROKEN - this page will test the cache filter is working correctly

Tag library examples

  • Jstl tags - shows the jstl basics
  • Custom tag test - show the basics of a custom tags in action
  • Custom tag file test - show the basics of a custom tag files in action

JavaBeans Examples

  • Get Property - a basic use of a simple java bean

JSP examples

  • Sandbox - this is a sample jsp so that we can play around with it
  • Include directive - this is an example of the way that includes can be used in a jsp
  • Applet Example - this is an example of to embed an applet into a JSP page
  • Request Headers - this is an example of how to show a requests headers
  • Show IP Address - shows the IP address of the calling client
  • Show Session - shows the session details

Servlet examples

  • Dynamic Image Servlet - simple servlet that creates a really simple image
  • Request Headers - simple servlet to show the request headers passed from the web browser
  • Example Form - to show you how to debug a forms parameter list
  • Show File Content - to show the content of a file upload request
  • File Upload - to show the actual file upload process from a servlet
  • Linked Servlets - to show how two servlets can be linked with request scope objects
  • Servlet Context (Mock Error) - to show how to pass servlet context through to the servlet
  • Show Web App Source - to show the source behind this web app
  • i18n Hello World - shows some examples of how to use internationalised character sets

Logging examples

  • Simple logging example - this will just drop some logging to the standard jdk logger
  • Site Logging Configuration - this will allow you to administer the current logging configuration
  • Test Logging page - this page allows you to test the logging at varying levels

Exception throwing examples

  • Throw Exception - this is an example of an exception being thrown.
  • Throw Exception to error page - this is an example of an exception being thrown and then managed by an error page.
  • Throw exception to to error servlet - this is an example of an exception being thrown and then managed by and error servlet.
  • Throw exception to generic exception handling - this is an example of an exception being thrown and managed by the web.xml configuration.
  • Throw exception for file not found - this should execute the file not found error page
